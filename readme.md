# Training Repository

## 01. Logic_Building

This directory contains a set of fundamental flowchart diagrams designed to explain basic programming logic, mathematical computations, series, and pattern generation using structured steps

*   **FlowCharts.md**: I created 12 flowchart diagrams to visualize problem-solving steps. These cover basic logic like finding the maximum of multiple numbers, mathematical series like Factorials and Fibonacci, and generating complex number patterns.

## 02. HTML-CSS

This directory contains project that demonstrates a responsive split-screen layout with a tabbed authentication interface, built using only HTML and CSS without any JavaScript.

## 03. MySQL

This dirctory contains task of SQL queries and their outputs. Its done using a movie-related dataset (movies, actors, directors, reviewers).

*   **Quries_with_Solution.md**: Contains 50 queries progressing from simple to complex.
    *   **Basics**: Simple selects, filtering by dates, and pattern matching.
    *   **Sub-Queries**: Nested requests to find specific data like actors in a certain movie.
    *   **Joins**: Linking multiple tables to connect directors, actors, and ratings in a single view.

## 04. JavaScript

This directory contains learnings for JavaScript programming, exercises focusing on JavaScript basic concepts. It contains 3 javascript tasks and solutions for exercism javascript track.

## 05. Angular

This directory contains projects created to learn angular core concepts. Almost all tasks and projects are done using 3 versions of Angular: 16,18,21. Each folder contains general readme file for task overview and then subfolders for specific version task details.
